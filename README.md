# [GSoC] McGill Space Institute: Pulsar's Data Visualization

## Setup

```
npm run setup
```

## Build & Use

```
npm run build
cd public
python -m SimpleHTTPServer # python 2
python -m http.server # python 3
```

## Libraries

* D3
