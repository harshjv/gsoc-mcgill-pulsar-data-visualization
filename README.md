# [GSoC] McGill Space Institute: Pulsar's Data Visualization


## Clone or download zip file

```
git clone https://github.com/harshjv/gsoc-mcgill-pulsar-data-visualization.git
```

## Setup & Build

```
npm install
```

## Run

```
node index.js
```

## Libraries

* D3
